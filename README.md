# Central Pet 🐾

Plataforma inteligente para adoção, resgate e proteção animal.

## Sobre

Central Pet é uma plataforma web que conecta ONGs, veterinários, petshops e tutores de animais para criar uma rede inteligente de proteção animal.

## Funcionalidades

- **Animais de Rua**: Cadastro de animais resgatados, pedidos de ajuda e adoção responsável
- **Mapa Pet**: Encontre petshops, clínicas, ONGs e casas de adoção perto de você
- **Pets Desaparecidos**: Sistema com IA para comparar animais encontrados e desaparecidos
- **Loja Pet**: Produtos para animais com parte da renda destinada ao resgate animal
- **SOS Resgate**: Alerta rápido para ONGs, veterinários e voluntários parceiros
- **Carteirinha Digital**: Vacinas, consultas e documentos salvos online com segurança

## Tecnologia

- React
- Tailwind CSS

## Como usar

1. Clone o repositório
2. Instale as dependências
3. Execute o projeto

```bash
npm install
npm run dev
```
